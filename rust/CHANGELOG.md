## [0.1.0] - Initial Release

### Added

- Initial implementation of LinoEnv struct for reading and writing .lenv files
- Support for `: ` (colon-space) separator as per links-notation specification
- Support for multiple values per key (duplicates allowed)
- Convenience functions `read_lino_env` and `write_lino_env`
- Full test coverage
- Documentation with examples
